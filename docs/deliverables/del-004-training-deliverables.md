- R2441 admin spec depends on operational guides (R2444) to exist.
- R2442 trainer spec mirrors R2443 end-user curriculum.
- R2443 end-user spec depends on R2442 trainers being certified
  before cohort delivery starts.

## 5. Acceptance criteria

- All four spec docs above exist and are signed by CNAS HR + supplier
  training lead.
- Attendance + assessment records produced per the umbrella plan §6
  (R2440).
- Bilingual delivery (RO/RU; RO/EN for admins) certified.
- Sign-off entered in the Acceptance Protocol rows "UTD 007 / R2441",
  "UTD 008 / R2442", "UTD 009 / R2443", "UTD 013 / R2444".
