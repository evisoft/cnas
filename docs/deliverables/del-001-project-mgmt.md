# DEL 001 — Project management deliverables (index)

> Anchored to TOR ID(s): R2600 (TOR §7.1). Index doc; content lives
> in the linked files. Iteration 103.

## 1. Purpose

Single navigation surface for the project-management deliverables
required by TOR §7.1 DEL 001: charter, plan, weekly reports, change
reports, phase reports, deviation reports.

## 2. Audience

CNAS Project Manager, supplier Project Manager, acceptance committee,
audit reviewers.

## 4. Acceptance criteria

- Every artefact above exists, is signed (where signature applies),
  and is referenced in the Acceptance Protocol (DEL 001 row).
- Weekly / phase / deviation reports filed at agreed cadence for the
  full delivery window.
- Index reviewed by CNAS PM and supplier PM jointly.

## 5. Status / open gaps

- Charter signature block: pending bilateral signature at contract
  signing (template ready, names blank).
- Weekly status reports: ongoing operational artefact, not authored
  in the repository.
- Phase / deviation reports: instantiated per event using the
  template; archive maintained outside the repo for confidentiality.
